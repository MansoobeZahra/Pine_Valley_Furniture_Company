# Internet Application Development Labs 1-7

This repository contains various lab assignments and projects developed for the Internet Application Development (IAD) course.

## Project Structure

- **index.html**: The main landing page to access all labs.
- **QuickByte**: A logistics and food delivery platform (Lab 10).
- **QuizWiz**: An online quiz management system (Lab 08).
- **FEEDBACK**: A survey and feedback system (Lab 09).
- **Labs 01-11**: Individual lab exercises covering HTML, CSS, JavaScript, and ASP.NET.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: ASP.NET (VB.NET), Web Forms
- **Database**: SQL Server (Hosted on Somee.com)

## How to Run

1. Open the project in Visual Studio or any ASP.NET compatible server.
2. Ensure the `Web.config` connection strings are correctly configured for your environment.
3. Start the application and navigate to `index.html`.
